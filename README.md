#  Purple Voca 

> **ChatGPT를 활용한 외국어 학습 확장 및 관리 서비스**
>
> 사용자가 직접 세운 목표 달성과 순위 시스템을 통해 성취감을 느끼며 효율적으로 영어를 학습할 수 있도록 돕는 안드로이드 앱입니다.

---

### 🌟 프로젝트 소개 (Project Introduction)
외국어 학습 확장에 도움이 되게끔 **ChatGPT**를 통해 맞춤형 영어 학습 계획을 작성하고, 이를 기반으로 영어 학습과 퀴즈를 진행합니다. 당일 배운 내용을 재학습할 기회를 제공하며, 자신의 목표 달성 여부와 실시간 순위 시스템을 통해 꾸준한 학습 동기를 부여합니다.

---

### 🛠 기술 스택 (Tech Stack)

| 구분 | 기술 |
| :--- | :--- |
| **Platform** | Android Studio |
| **Language** | JAVA |
| **AI Integration** | ChatGPT API |
| **Database** | Firebase (또는 사용하신 DB 명칭) |

---

### 🚀 주요 기능 (Key Features)
* **AI 학습 계획:** ChatGPT를 이용한 개인별 맞춤 학습 스케줄링
* **학습 및 퀴즈:** 계획 기반의 영어 공부와 자가 진단 퀴즈
* **복습 시스템:** 당일 학습 내용에 대한 재학습 및 피드백
* **성취감 부여:** 목표 달성 체크 및 사용자 간 순위 시스템
